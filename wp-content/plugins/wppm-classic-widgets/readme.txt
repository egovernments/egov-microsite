== Description ==

Using WPPM Classic Widgets, you can restores the previous ("classic") WordPress widgets settings screens in Appearance.

Once you wil activate this plugin, you will get the previous widgets settings screens and disables the block editor from managing widgets. You don't need to do any configuration and setting for this, Just install the plugin and activate it as simple at usual.

== Installation ==

1. Upload the directory `/wppm-classic-widgets/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now check your widget screens in Appearance
